# Features By Module

This document organizes Life OS capabilities by module and clarifies what belongs in MVP versus later releases.

## Today OS

### Purpose

Help the user win the current day with clarity, priorities, and next actions.

### MVP

- Daily summary header
- Daily score or momentum score
- Top 3 priorities
- Lightweight tasks and reminders
- Routine progress
- Incomplete items
- Health basics snapshot
- Spending snapshot
- Attention panel
- Quick capture entry point

### Later phases

- Dynamic recommendations
- Smarter day balancing based on sleep, recovery, and workload
- Calendar integration

## Habit OS

### Purpose

Build consistency through fast completion, visible streaks, and routine support.

### MVP

- Habit creation and editing
- Categories
- One-tap completion
- Morning and evening routine grouping
- Streaks
- Weekly consistency summary

### Later phases

- Identity-based habit framing
- Habit impact correlations
- Adaptive habit targets

## Health OS

### Purpose

Track a small set of important physical signals that influence daily readiness.

### MVP

- Water logging
- Meal logging with templates
- Workout status tracking
- Body weight logging
- Basic daily and weekly summaries

### Later phases

- Macro and calorie depth
- Sleep logging or integrations
- Recovery scoring
- Body progress media or measurement support

## Finance OS

### Purpose

Give the user lightweight money visibility without complex financial tooling.

### MVP

- Expense logging
- Spend categories
- Period spend summary
- Recurring expense templates
- Upcoming bill reminders

### Later phases

- Savings goals
- Budget guardrails
- Subscription management depth
- Import or account sync

## Planning OS

### Purpose

Turn life goals into weekly and daily direction.

### MVP

- Goal capture
- Weekly priorities
- Monthly focus
- Goal references in review flow
- Simple linkage between goals and daily priorities

### Later phases

- Projects
- Milestones
- Quarterly planning
- Roadmap views across life domains

## Reflection OS

### Purpose

Help the user learn from patterns and improve the system over time.

### MVP

- Daily review flow
- Weekly review flow
- Monthly review flow
- Wins, misses, lessons
- Basic scorecard summaries
- Next-week planning output

### Later phases

- Richer prompt libraries by cadence
- Guided retrospectives
- Reflection history and pattern extraction

## Life Admin OS

### Purpose

Keep mundane but important life maintenance from derailing the rest of the system.

### MVP

- Not included as a standalone module
- Only lightweight reminders supported through quick capture and recurring items

### Later phases

- Grocery lists
- Chores
- Home maintenance
- Errands
- Document reminders

## Insights and Attention Engine

### Purpose

Summarize what matters and call out drift before it becomes a bigger problem.

### MVP

- Rule-based attention cards
- Rule-based daily score
- Momentum and streak feedback tied to meaningful actions
- Weekly summary patterns
- Transparent alert reasons

### Later phases

- Correlation views
- Balance meter
- Richer scoring and readiness score
- More personalized recommendations
