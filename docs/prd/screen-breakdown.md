# Screen Breakdown

This document translates the product scope into concrete screens and key UI responsibilities.

## 1. Home

### Purpose

Serve as the primary daily command center.

### Core content

- Greeting and date
- Daily summary or readiness area
- Daily score or momentum score
- Top 3 priorities
- Lightweight tasks and reminders
- Routine progress
- Health basics snapshot
- Spending snapshot
- Attention panel
- Quick capture button

### Primary actions

- Open Today planning view
- Complete a habit or routine item
- Log a quick action
- Review what needs attention

### MVP status

Required

## 2. Today

### Purpose

Provide a focused execution view for the current day.

### Core content

- Priorities
- Optional time blocks or plan list
- Tasks or reminders
- Meal and workout plan status
- Notes

### Primary actions

- Reorder priorities
- Mark completion
- Carry forward unfinished items

### MVP status

Required

## 3. Habits

### Purpose

Manage habits, routines, and consistency.

### Core content

- Habit list
- Streaks
- Completion states
- Morning and evening routines
- Weekly consistency summary

### Primary actions

- Add or edit habits
- Complete habits
- Review habits at risk

### MVP status

Required

## 4. Health

### Purpose

Capture and review daily physical inputs that matter to consistency.

### Core content

- Water progress
- Meals logged
- Workout status
- Body weight trend

### Primary actions

- Log water
- Log meal
- Mark workout status
- Add body weight

### MVP status

Required

## 5. Finance

### Purpose

Provide simple visibility into spending and upcoming bills.

### Core content

- Current period spend
- Category summaries
- Recent expenses
- Recurring payments or upcoming bills

### Primary actions

- Add expense
- Edit expense
- Review category spend

### MVP status

Required

## 6. Goals and Planning

### Purpose

Store longer-term direction and translate it into weekly focus.

### Core content

- Goals by life area
- Weekly priorities
- Monthly focus
- Simple links from goals to current focus

### Primary actions

- Add or edit goals
- Set weekly priorities
- Reference goals during review

### MVP status

Required, but lightweight

## 7. Review

### Purpose

Support daily, weekly, and monthly reflection and planning.

### Core content

- Daily review mode
- Habit summary
- Health basics summary
- Spending summary
- Weekly review mode
- Monthly review mode
- Wins, misses, lessons prompts
- Next-week planning section
- Monthly focus section

### Primary actions

- Complete review by cadence
- Set weekly targets
- Set monthly focus
- Save lessons learned

### MVP status

Required

## 8. Quick Capture

### Purpose

Make common inputs fast from any context.

### Core content

- Event type selector
- Short form by event type
- Recent templates
- Defaults and recurring options

### Supported input types in MVP

- Task
- Expense
- Water
- Meal
- Workout status
- Weight
- Note
- Reminder

### MVP status

Required

## 9. Life Admin

### Purpose

Eventually manage recurring life-maintenance work.

### MVP status

Deferred

## 10. Insights

### Purpose

Eventually provide deeper trend analysis and cross-domain visibility.

### MVP status

Deferred as a standalone screen

### MVP replacement

Short attention cards, scoring feedback, and weekly summaries embedded in Home and Review
