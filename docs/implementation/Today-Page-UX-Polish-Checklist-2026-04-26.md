# Today Page UX Polish Checklist

## Goal
Make the Today page easier to read, understand, and use throughout the day without changing the core two-column structure.

## Issues To Fix
- [x] Improve contrast for muted metadata such as `Today`, `Next action`, task times, goal detail text, and small labels.
- [x] Strengthen the selected task panel so actions feel intentional instead of floating text.
- [x] Reduce the right column’s “stacked signals” feeling so support sections feel guided, not like a dense system list.
- [x] Clarify `Goal Momentum`; the current wording does not immediately explain why the suggestion matters today.
- [x] Make Daily Essentials affordances clearer if rows are clickable.
- [x] Tighten vertical spacing in the task queue so calmness does not become wasted space.
- [x] Move `Plan these` closer to the `Unplanned` section so the action clearly belongs to that group.
- [x] Reposition or restyle `3/9 done` so progress is easier to notice with the task queue title.
- [x] Remove repeated `Needs recovery` text from every overdue task and replace it with a calmer section-level signal.

## Non-Goals For This Pass
- Do not redesign the full Today page again.
- Do not rebuild the future Rescue Mode workflow.
- Do not change the two-column layout.
- Do not add new major features.

## Later Improvements
These are valid ideas, but they are intentionally deferred until after the current polish pass.
