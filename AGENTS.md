## Skills

### Available skills

- `frontend-copilot-cli`: Delegate all frontend implementation, editing, debugging, refactoring, and UI/UX design work in this repository to GitHub Copilot CLI. Use whenever a task touches `client/`, React components, routes, CSS, frontend state, accessibility, visual design, or mixed frontend/backend changes where Codex should complete backend work locally and then hand the frontend scope to Copilot CLI. (file: `/home/ubuntu/Development/life-os/agents/skills/frontend-copilot-cli/SKILL.md`)
- `frontend-design`: Create distinctive, production-grade frontend interfaces with high design quality. Use together with `frontend-copilot-cli` when a frontend task needs stronger visual direction or higher design polish. (file: `/home/ubuntu/Development/life-os/agents/skills/frontend-design/SKILL.md`)

### How to use skills

- If a task is frontend-related, use `frontend-copilot-cli`.
- If the frontend task also needs stronger visual direction, use `frontend-design` first for design direction and then execute through `frontend-copilot-cli`.
- Read only enough of each `SKILL.md` to follow the workflow.
- Load extra reference files only when the active skill points to them.
